How to use Embed Image Downloader?

  1. First, Go to Visual paradigm website: Link to Website.
  2. Choose the diagram you want to create and finish that diagram.
  3. To Export: Click on File > Embed > Image and Embed.
  4. Now, Copy the link and paste it in Embed Image Downloader.
  5. Click fetch image and Download the image.
